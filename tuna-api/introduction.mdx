## Nedir?

Üretimi yapılan kitapların kutulama ve paletleme işlemleri sonrasında alıcı firma tarafından kendi sistemlerine kitap barkod, bandrol ve ISBN numaralarının kolayca entegre edilmesi sağlamak amacıyla geliştirilmiştir.

## Nasıl Çalışır?

Üretim esnasında robotik kontrol sistemleri tarafından üretilen ürünlerin bilgileri otomasyon ile sistemimize kaydedilir. Kaydedilen bu bilgiler alıcı firma tarafından yapılan API isteğiyle karşıya iletilir.

## İhtiyaçlar

Firma bilgilerinizi ileterek üretici firmadan kendinize ait API erişim bilgilerini almalısınız. Ayrıca erişim yapacağınız IP adreslerini iletmelisiniz. Bu bilgiler iletildikten sonra tarafınıza kullanıcı adı ve parola bilgilendirmesi yapılacaktır.
